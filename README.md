# AI Financial News Summarizer

## Overview
This project fetches live financial news using NewsAPI and processes the results using Python and Pandas.

## Features
- Fetches real-time financial news
- User enters a company name
- Displays latest headlines
- Stores news in CSV format
- Built using Python, Pandas and APIs

## Technologies Used
- Python
- Pandas
- NewsAPI
- VS Code

## Sample Output

The project generates:
- News headlines
- Source information
- CSV export file
